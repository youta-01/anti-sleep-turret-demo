# ADR 001: アラームだけでなく入浴を介入にする

## Context

作者は、通常のアラームだけの方針と区別でき、開始と完了を明示できる介入を必要条件として定義しました。

## Decision

眠気リスクが方針上の条件を満たしたとき、入浴の開始と完了を状態機械で扱うBath介入を中心にします。

## Alternatives

- 音声・画面アラームだけを続ける
- 即時に作業を完全停止する
- 介入を記録だけに留める

## Trade-offs

行動として明確ですが、時間と負担が大きく、例外設計と安全境界が不可欠です。

## Evidence boundary

公開デモはBath介入の状態遷移を合成イベントで検証します。実際の行動変化、因果効果、効果量は公開版の検証対象ではありません。

## Current status

採用中。作者本人向けのN=1方針であり、医療・一般利用を対象にしません。
