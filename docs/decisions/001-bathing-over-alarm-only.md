# ADR 001: アラームだけでなく入浴を介入にする

## Context

作者の実運用では、通常のアラームは停止できても、その後の行動変化につながらない場合がありました。目的はCG制作へ使える時間を守ることです。

## Decision

眠気リスクが方針上の条件を満たしたとき、入浴の開始と完了を状態機械で扱うBath介入を中心にします。

## Alternatives

- 音声・画面アラームだけを続ける
- 即時に作業を完全停止する
- 介入を記録だけに留める

## Trade-offs

行動として明確ですが、時間と負担が大きく、例外設計と安全境界が不可欠です。

## Observed consequence

記録された事例では、Ast作動後に実際の入浴行動へ至ったことがあります。因果効果や効果量は示しません。

## Current status

採用中。作者本人向けのN=1方針であり、医療・一般利用を対象にしません。
