# 公開版と非公開版の境界

## 公開版に含むもの

- dataclassとenumによる依存なしのドメインモデル
- 制御可能なFake Clock
- 合成センサー、合成QR、合成充電・Presenceイベント
- 現行のBath、Riz、例外モードを表す純粋ロジック
- TkinterシナリオUI、ヘッドレスCLI、テスト、公開前スキャン
- サニタイズ済みの実プロトタイプ画面と動画

## 公開版に含めないもの

- 実カメラ・画像認識処理
- 実スマートフォン自動化、通信経路、認証情報、QR内容
- ローカル設定、実行時記録、個人の自己管理データ
- Windowsプロセス制御やTechnical Safety Cutoffの具体的な停止手順
- 非公開プロトタイプのソースコード

## 廃止した設計

以前の版では Return to PC と Recovery Work を試しました。非公開プロトタイプには休眠中の互換コードとして残っていますが、現行の標準フローおよび公開デモには含みません。Bath完了後は直接 `MONITORING` に戻ります。

公開版は非公開実装のコピーではなく、公開を承認した事実と現行の有効仕様から再構成したポートフォリオ用モデルです。
