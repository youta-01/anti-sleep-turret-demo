# 公開版と非公開版の境界

## 公開版に含むもの

- dataclassとenumによる依存なしのドメインモデル
- 制御可能なFake Clock
- 合成センサー、合成QR、合成充電イベント
- 現在有効な入浴、Riz、例外モードの振る舞い
- TkinterシナリオUI、ヘッドレスCLI、テスト、公開前スキャン

## 公開版に含めないもの

- 実カメラおよび画像認識処理
- 実スマートフォンオートメーションと通信経路
- 認証情報、QR内容、ローカル設定、実行時記録
- Windowsプロセス制御
- 個人の生データや詳細な個人記録

## 廃止した実験

非公開側には、開発過程で試した `RETURN_TO_PC`、`RECOVERY_WORK`、`SUCCESS_COOLDOWN`、`BATH_READY` が休眠コードとして残っています。現在の有効動作ではなく、公開enum、遷移、UI操作には含めていません。入浴完了は直接 `MONITORING` に戻ります。

公開版は非公開実装のコピーではなく、公開を承認した事実と現在有効な仕様から再構成したポートフォリオ用モデルです。
