# 実運用上の観察と限界

## 承認された観察

記録された事例では、Astの作動後に実際の入浴行動へ至ったことがあります。これは「作動と行動が同じ記録内で観察された」という範囲の記述であり、効果の原因を確定するものではありません。

## 現在の限界

現在の自己管理データは大部分が後ろ向きです。因果的な有効性、効果量、異なる日種別にわたる一般的な信頼性を示すものではありません。医学的有効性、睡眠診断、一般利用者への有効性、定量的な生産性向上も主張しません。

## 今後の検証

- AstイベントのメタデータをNotion / CLL-Nへ自動送信する
- 平日、休日、アルバイト日、イベント日を比較する
- 外乱と環境変化を評価する
- 変数の有効期間を定義する
- 後ろ向き記録ではなく前向き記録を使う

今後も、観察、仮説、介入、結果を分離して記録し、確認できる範囲を越えて結論を広げないことを優先します。
